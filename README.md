# Online Pet Item Purchasing

**React + Node + MongoDB based Online Pet Item Purchasing website.**  
Includes Admin, Supplier, and Customer management.

---

## Project Overview

An e-commerce platform for pet supplies where:

- **Customers** can browse products, add to cart and place orders.
- **Suppliers** can add and manage products.
- **Admin** can approve suppliers, manage users, and view reports.

---

## Features

- User authentication (customers & suppliers).
- Product listing, search and categories.
- Cart and checkout flow.
- Order management and invoice generation.
- Admin dashboard (approve suppliers, manage orders).
- REST API built with Node.js + Express and MongoDB.

---

## Tech Stack

- Frontend: **React**
- Backend: **Node.js, Express**
- Database: **MongoDB**
- Version control: **Git & GitHub**
